VIM\_IDE
======
This is a vim configuration files project.
Powered By [Vundle](http://github.com/gmarik/vundle) and [dot-vimrc](https://github.com/humiaozuzu/dot-vimrc)
to install this

``` sh
#clone
git clone git@github.com:LanderlYoung/vim_ide.git <somewhere>
cd <somewhere>
#setup
./install.sh
```

then all done!

have fun!
